# Less

# 第一章 Less介绍

# 第一节 CSS中存在的问题

CSS 是一门描述语言，缺少灵活性和复用性。

# 第二节 常用CSS预处理工具介绍

  1. Less 诞生于 2009 年，受Sass的影响创建的一个开源项目。 它扩充了 CSS 语言，增加了诸如变量、混合（mixin）、函数等功能，让 CSS 更易维护、方便制作主题、扩充（引用于官网）。
  2. Sass 诞生于 2007 年，Ruby 编写，其语法功能都十分全面，可以说 它完全把 CSS 变成了一门编程语言。另外 在国内外都很受欢迎，并且它的项目团队很是强大 ，是一款十分优秀的预处理语言。
  3. Stylus 诞生于 2010 年，来自 Node.js 社区，语法功能也和 Sass 不相伯仲，是一门十分独特的创新型语言。

# 第三节 使用Gulp自动化构建less

const gulp = require('gulp')
const less = require('gulp-less')

gulp.task('css', done => {
    gulp.src('./src/**/*.less')
        .pipe(less())
        .pipe(gulp.dest('./dist/'))
    done()
});

gulp.task('watch', done=>{
    gulp.watch('./src/**/*.less', gulp.series('css'))
    done();
})

# 第二章 Less 扩展语法

# 第一节 注释

CSS 形式的注释在 LESS 中是依然保留的:

/* Hello, I'm a CSS-style comment */
.class { color: black }
1
2

LESS 同样也支持双斜线的注释, 但是编译成 CSS 的时候自动过滤掉:

// Hi, I'm a silent comment, I won't show up in your CSS
.class { color: white }

2

# 第二节 变量

变量允许我们单独定义一系列通用的样式，然后在需要的时候去调用。所以在做全局样式调整的时候我们可能只需要修改几行代码就可以了。

// 主色调
@primary-color: #5B83AD;

#header {
  color: @primary-color;
}

# 第三节 导入

你可以在main文件中通过下面的形势引入 .less 文件, .less 后缀可带可不带:

@import "lib.less";
@import "lib";
1
2

如果你想导入一个CSS文件而且不想LESS对它进行处理，只需要使用.css后缀就可以:

@import "lib.css";
1

这样LESS就会跳过它不去处理它.

变量和导入的实现使得模块化成为可能。

# 第四节 嵌套

我们可以在一个选择器中嵌套另一个选择器来实现继承，这样很大程度减少了代码量，并且代码看起来更加的清晰。

#header {   
	h1 {     
		font-size: 26px;     
		font-weight: bold;  
	}   
	p { 
		font-size: 12px;     
		a { 
			text-decoration: none;       
			&:hover { border-width: 1px }     
		}   
	}
}

# 第五节 混合 Mixins

  • 变量提供了数据的复用
  • 混合提供了样式集的复用
.rounded-corners  
{   
    border-radius: 5px;
}  
#header {
    .rounded-corners;
}

# 第三章 值处理

# 第一节 运算

任何数字（时间、角度、长度）、颜色、%百分比、变量都可以参与 +, -, *, /运算. 来看一组例子:

@base: 5%;
@filler: @base * 2;
@other: @base + @filler;

color: #888 / 4;
background-color: @base-color + #111;
height: 100% / 2 + @filler;
1
2

4

LESS 的运算已经超出了我们的期望，它能够分辨出颜色和单位。如果像下面这样单位运算的话:

@time: 1s + 500ms; // 输出1.5s
@var: 1px + 5; // 输出 6px
1
2

括号也同样允许使用:

width: (@var + 5) * 2;
1

并且可以在复合属性中进行运算:

border: (@width * 2) solid black;
