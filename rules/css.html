                # CSS 编码规范

                # 缩进

                • 使用soft tab（4个空格）。

                # 分号

                • 每个属性声明末尾都要加分号,最后一个属性也要加分号

                # 空格

                • 以下几种情况需要空格：
                  • 属性值前
                  • 选择器'>', '+', '~'前后
                  • '{'前
                  • !important '!'前
                  • @else 前后
                  • 属性值中的','后
                  • 注释'/'后和'/'前
                .element {
    color: red !important;
    background-color: rgba(0, 0, 0, .5);
}

.element > .dialog{
    ...
}

                # 换行

                • 以下几种情况不需要换行：

                  • '{'前

                • 以下几种情况需要换行：

                  • '{'后和'}'前
                  • 每个属性独占一行
                  • 多个规则的分隔符','后
                /* 不好 */
.element
{color: red; background-color: black;}

/* 好 */
.element {
    color: red;
    background-color: black;
}

/* 不好 */
.element, .dialog {
    ...
}

/* 好 */
.element,
.dialog {
    ...
}

                # 命名

                • 类名使用小写字母，以中划线分隔
                • id采用驼峰式命名
                /* class */
.element-content {
    ...
}

/* id */
#myDialog {
    ...
}

                # 属性声明顺序

                • 相关的属性声明按右边的顺序做分组处理，组之间需要有一个空行。
                .declaration-order {
    display: block;
    float: right;

    position: absolute;
    top: 0;
    right: 0;
    bottom: 0;
    left: 0;
    z-index: 100;

    border: 1px solid #e5e5e5;
    border-radius: 3px;
    width: 100px;
    height: 100px;

    font: normal 13px "Helvetica Neue", sans-serif;
    line-height: 1.5;
    text-align: center;

    color: #333;
    background-color: #f5f5f5;

    opacity: 1;
}

                # 颜色

                • 颜色16进制用小写字母；

                • 颜色16进制尽量用简写。

                /* 不好 */
.element {
    color: #ABCDEF;
    background-color: #001122;
}

/* 好 */
.element {
    color: #abcdef;
    background-color: #012;
}
